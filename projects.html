Weather Application
Click here for WeatherApp Github Repository
Description

This Java-based project retrieves current weather data for a user-specified location using the WeatherStack API. Upon running the program, the user is prompted to enter the name of a city, and the program constructs a dynamic URL to query the weather API. The application sends an HTTP GET request to the API, which returns a JSON response containing weather information. The program then reads and displays this response. Error handling is built into the application to check for missing API keys and unsuccessful API requests. The use of an environment variable for the API key enhances security by keeping the key out of the source code. This is an ongoing project.

Key Features

Key Features

  • Weather API Integration
  • User Input Handling
  • Error Handling
  • Scalability
  • Response Parsing
  • Web Interface
Further Steps

I am currently working on fully integrating the JSON parser to efficiently extract and display relevant weather details. Currently, the program simply prints the raw JSON response to the console, but the next step would be to parse the JSON data and display specific weather information. Additionally, I am working towards transforming this project from a command-line interface (CLI) to a web interface, which will feature input fields for location queries, a display area for the weather data, and other user-friendly elements.

Calculator
Click here for Calculator Github Repository
Description

This Java-based calculator program is designed to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The program processes the user input by splitting it into tokens, distinguishing between numbers and operators. It then uses a series of modular functions to execute the appropriate arithmetic operation. The program supports both integers and floating-point numbers and performs operations sequentially based on the input. Error handling is included to catch invalid number formats, division by zero, and other potential issues, with custom error messages displayed for each case. This is an ongoing project.

Key Features
  • Basic Calculator Operation
  • Supports Multiple Operations in Sequence
  • Number Type Flexibility
  • User Input Handling
  • Error Handling
  • GUI Interface
Further Steps
